Corona App






To access specific states append the 'states' query parameter to the end of the URL and add the states symbols.

Examples:

http://www.gpc4j.org/corona/index.xhtml?states=or,mi,tx,wi,ca,wa

http://www.gpc4j.org/corona/index.xhtml?states=ca,wa,ny

To exclude specific states append the 'exclude' query parameter to the end of the URL and add the states symbols.

Example:

http://www.gpc4j.org/corona/index.xhtml?exclude=ny,wa

To access specific charts use the corresponding links below and append the 'states' or 'exclude' query parameters to the end of the URL as above if wanted.

Examples:

http://www.gpc4j.org/corona/region.xhtml

http://www.gpc4j.org/corona/active.xhtml

http://www.gpc4j.org/corona/deaths.xhtml
